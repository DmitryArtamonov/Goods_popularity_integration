Программа берет данные о количестве продаж каждого товара за определенный период
в Мой Склад и записывает их в параметр 'sort_weight' в Insales. Этот параметр
может использоваться для сортировки по популярности. В отличии от встроенной в Insales
сортировки: а) учитывается количество продаж, а не количество товара; б) берутся данные
за период, а не за все время работы магазина.

Алгоритм:
1. Формируем словарь {'insales_id':{'sku'}} из Инсейлс
2. Выгружаем товары из МС и добавляем в словарь 'ms_id', связываем по sku. Сообщаем, если
sku дублируется
3. По каждому товару в словаре выгружаем количество отгрузок за период из МС. Добавляем в словарь
в поле 'orders'.
